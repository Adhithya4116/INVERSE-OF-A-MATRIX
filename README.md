## Program:
```#Program to find the inverse of a matrix.
#Developed by: adhithya perumal
#RegisterNumber:22008747
import numpy as np
A=np.array([[2,1,1],[1,1,1],[1,-1,2]])
result=np.linalg.inv(A)
print(result)
```
